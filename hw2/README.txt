How to run my program
1. Go to the src folder in your terminal. (requirements.txt file is in the src folder)
2. pip install -r requirements.txt
3. python main.py

*****NOTICE*****
You can see some plot during run my progroam.

You cloese all the plots then you can see my model.summary and the result including accuracy and loss.
