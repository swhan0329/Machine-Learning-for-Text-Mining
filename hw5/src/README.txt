./hw5.sh runs CTF and DF.
But dev-predictions.txt and test-predictions.txt are stored for the DF.

If you want to view the result of the CTF, JUST USE COMMAND BELOW:
python main.py 0

If you want to view the result of the DF. JUST USE COMMAND BELOW:
python main.py 1

SVM.py is a file that helps to preprocess liblinear. 
This file handles all of the CTF and DF at once.
If you want to run SVM.py, USE COMMAND BELOW:
python SVM.py
